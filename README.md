## 🦸‍♂️ Instalación y Uso para Humanos (¡Cero Excusas!)

¿Nunca usaste Node.js? ¿Te asusta la terminal? ¡No importa! Aquí te lo explico como si tuvieras 5 años (o 95).

### 1. Instala Node.js (¡1 solo click!)

- Ve a 👉 [nodejs.org](https://nodejs.org/) y descarga la versión **LTS** (recomendada para todos).
- Haz doble click en el instalador y sigue el clásico “Siguiente, siguiente, siguiente…”.
- ¡Listo! Ya tienes superpoderes.

### 2. Descarga OPEN AUDITOR

- Haz click en el botón verde “Code” arriba a la derecha en GitHub y elige “Download ZIP”.
- O si te animas, usa la terminal:
  ```bash
  git clone https://github.com/tu-usuario/open-auditor.git
  ```

### 3. Abre la Terminal (¡No muerde!)

- **Windows:** Busca “Terminal” o “Símbolo del sistema” en el menú inicio.
- **Mac:** Abre “Terminal” desde Aplicaciones > Utilidades.
- **Linux:** Ctrl+Alt+T y ya.

### 4. Instala y Lanza la App

En la terminal, escribe:
```bash
cd open-auditor
npm install
npm start
```
Abre tu navegador en 👉 **http://localhost:3001**

## 🧯 Solución de Problemas Ultra-Rápida

**¿No abre la app?**
- ¿Node.js instalado? Prueba `node -v` en la terminal.
- ¿Puerto 3001 ocupado? Cierra otros programas o cambia el puerto en `server.js`.
- ¿Error raro? Escribe el mensaje en Google o abre un [Issue](https://github.com/tu-usuario/open-auditor/issues).

**¿Quieres actualizar la app?**
```bash
git pull
npm install
```

**¿Quieres respaldar tus datos?**
- Haz copia de la carpeta `data/` y guárdala donde quieras. ¡Tus inspecciones viven ahí!

## 🤔 Preguntas Frecuentes (FAQ)

**¿Funciona en Windows/Mac/Linux?**
Sí, en todos. Solo necesitas Node.js.

**¿Necesito internet?**
Solo para descargar la app la primera vez. Después, 100% offline.

**¿Mis datos están seguros?**
Totalmente. Todo queda en tu computadora, en la carpeta `data/`.

**¿Puedo usarlo en una tablet?**
¡Sí! Abre la app desde el navegador de tu tablet conectada a la misma red.

**¿Puedo perder mis inspecciones?**
Solo si borras la carpeta `data/`. Haz backups regulares.

**¿Puedo tener varias instalaciones?**
Sí, pero cada una tendrá sus propios datos.

**¿Puedo migrar mis datos a otra PC?**
Copia la carpeta `data/` a la nueva máquina. ¡Y listo!

## ⚡ Quick Start: Auditor en 60 Segundos

Necesitas [Node.js](https://nodejs.org/) (v18+) y listo. Abre tu terminal y... ¡acción!

```bash
# 1. Clona este repositorio (una sola vez)
git clone https://github.com/tu-usuario/open-auditor.git

# 2. Entra en la carpeta

# 3. Instala las dependencias (ultra rápido)
npm install

# 4. ¡Lanza el cohete! 🚀
```

Abre tu navegador en 👉 **[http://localhost:3001](http://localhost:3001)**

¡Y ya está! Estás viendo el panel de control. El poder está en tus manos.

## 🛠️ Stack Tecnológico: Minimalismo y Potencia Pura

Construido sobre la filosofía de "menos es más". Sin frameworks pesados, sin dependencias inútiles.

- **Frontend:** HTML + CSS + JavaScript (Vanilla JS). Carga instantánea.
- **Backend:** Node.js + Express. Ligero y robusto.
- **"Base de Datos":** Archivos JSON. Portables, legibles y fáciles de respaldar.
- **Motor de Reportes:**
  - **PDF:** Puppeteer (usa Chromium para PDFs perfectos).
  - **Excel:** ExcelJS (crea `.xlsx` nativos).
