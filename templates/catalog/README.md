# Community Template Catalog

Este directorio contiene plantillas comunitarias listas para importar y adaptar dentro de OPENAUDITOR.

## Orientación normativa argentina

Estas plantillas están pensadas como base operativa para inspecciones alineadas con:

- Ley 19.587 de Higiene y Seguridad en el Trabajo
- Decreto 351/79 como reglamentación general
- Resolución SRT 299/2011 para criterios de EPP, cuando corresponda
- Decreto 911/96 para plantillas de construcción

No reemplazan asesoramiento legal ni técnico profesional. Sirven como punto de partida para que cada organización ajuste sus listas según actividad, jurisdicción, ART, convenio y riesgos propios.

## Criterios recomendados

- descripción clara y reutilizable
- preguntas accionables
- portada con datos mínimos de contexto
- estructura fácil de adaptar
- tags por rubro, uso y tipo de inspección
- referencia normativa para contextualizar el checklist

## Enfoque sugerido

Para Argentina, el catálogo apunta a que cualquier equipo pueda arrancar con una base seria y luego adaptarla según:

- actividad real del establecimiento
- riesgos específicos
- exigencias de ART
- políticas internas
- normativa complementaria aplicable

Todas se pueden importar desde la UI del dashboard o usando los endpoints de ejemplos del catálogo.
