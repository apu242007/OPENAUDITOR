OPEN AUDITOR
Repositorio README
Auditorias e inspecciones

Plantillas, ejecucion y reportes PDF en un solo flujo.

OPEN AUDITOR permite crear plantillas de auditoria, iniciar inspecciones, registrar evidencia y exportar reportes con branding propio. Esta pagina es la portada estatica para GitHub Pages.

Ver codigo Como ejecutarlo
Incluye

Editor de plantillas

Paginas, secciones, preguntas, logica condicional y versionado.

Salida

PDF con logo

Reportes con branding, portada, evidencias, firma y hallazgos.

Importante

GitHub Pages es estatico

La app completa necesita el backend Node.js para funcionar localmente o en un servidor.

Crear

Disena auditorias con portada, preguntas puntuables, adjuntos y acciones correctivas.

Inspeccionar

Ejecuta inspecciones desde el navegador, guarda progreso y registra evidencia multimedia.

Exportar

Genera PDFs, hojas Excel, CSV y paquetes reutilizables para compartir plantillas.

Para usar la aplicacion completa

Clona el repositorio, instala dependencias y ejecuta el servidor Node.js. GitHub Pages mostrara esta portada, pero no puede reemplazar las rutas API del backend.

npm install && npm start
