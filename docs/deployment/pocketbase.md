# PocketBase

PocketBase es la opción open source recomendada para dar un siguiente paso sin encerrar el proyecto en un SaaS.

## Uso Sugerido

- OPENAUDITOR sigue siendo la herramienta de diseño y administración
- el móvil usa HTML autónomo
- el HTML sincroniza a un endpoint propio que guarda en PocketBase

## Qué Ganás

- auth simple
- storage integrado
- despliegue liviano
- colección de imports y adjuntos

## Colecciones Sugeridas

- `inspections_imports`
- `inspection_media`
- `devices`
- `operators`
