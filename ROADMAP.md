# ROADMAP

## P1 - Adopcion Open Source

- guias de despliegue listas para usar
- documentacion para integradores
- payloads y esquemas documentados
- estrategia offline/sync mas visible
- changelog utilizable
- catalogo comunitario mas fuerte y visual

## P2 - Operacion y Confianza

- mas cobertura de tests
- backups programados
- mejores validaciones de import/export
- auditoria de cambios
- menos deuda tecnica en frontend
- CSP mas estricto sin romper UX

## P3 - Ecosistema

- conectores y sistema de extensiones
- mas destinos de sync listos para usar
- internacionalizacion completa
- instaladores para usuarios no tecnicos
- paquetes sectoriales por rubro y por pais

## P4 - Escalado Open Source

- sincronizacion remota bidireccional opcional
- dashboards y analitica
- backend opcional mas modular
- experiencia PWA mas madura
- despliegue simple para equipos distribuidos
