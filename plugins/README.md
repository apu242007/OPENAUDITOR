# Plugins and Connectors

Esta carpeta deja una base para futuras extensiones del proyecto.

## Objetivo

Permitir que la comunidad construya conectores sin tocar el núcleo para cada caso.

## Ideas de Plugins

- exportadores
- importadores
- conectores ERP
- conectores BI
- receptores de sync
- transformadores de payload

## Estado Actual

Todavía no hay un sistema formal de plugins cargables. Esta carpeta funciona como lugar de diseño, ejemplos y contratos de integración.
